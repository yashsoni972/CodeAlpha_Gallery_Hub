## ✨ Features

- 📁 **Upload Photos** — Drag & drop or browse to upload images (JPG, PNG, WEBP, GIF)
- 🏷️ **Auto Category Detection** — Automatically detects category from filename keywords
- 🗂️ **Filter by Category** — Tabs for Nature, Architecture, Travel, Portrait, Abstract, Food + custom categories
- 🔲 **Grid / List View** — Toggle between a masonry-style grid and a list layout
- 🔍 **Lightbox Viewer** — Full-screen image viewer with prev/next navigation and keyboard shortcuts
- ⬇️ **Download Images** — Download any photo directly from the lightbox
- 🗑️ **Delete Cards** — Remove any image from the gallery with a smooth animation
- ➕ **Custom Categories** — Add your own category during upload; it persists as a new chip and filter tab
- 📊 **Live Stats** — Real-time photo count and uploaded count in the hero section
- 📱 **Fully Responsive** — Works seamlessly on desktop, tablet, and mobile

## 🚀 Getting Started

No build tools or dependencies needed — it's pure HTML, CSS, and JS.

```bash
# Clone the repository
git clone https://github.com/yashsoni972/CodeAlpha_Gallery_Hub.git

# Open in browser
cd CodeAlpha_Gallery_Hub
open index.html
```

Or simply open `index.html` directly in any modern browser.
